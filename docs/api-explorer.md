# API Explorer

The interactive explorer is available here:

- [Open the API explorer](/api)

## What it does

- loads the repo OpenAPI file directly
- lets you search endpoints and operations quickly
- shows parameters, schemas, request bodies, and responses
- keeps the docs site separate from the machine-generated explorer UI

## Why this layout

The GameBanana spec is large and changes often. Keeping the guides in normal Docusaurus docs and the API viewer in a dedicated explorer page makes the site easier to maintain while still giving you a searchable OpenAPI interface.
